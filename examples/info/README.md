# Hello

Print current WGPU version and selected adapter information.

```shell
odin build ./info -out:./build/<executable-name>
```

Output example:

```shell
WGPU version: 0.17.0.2

Selected device:

NVIDIA GeForce RTX 3060
        Driver: 536.99
        Type: Discrete GPU with separate CPU/GPU memory
        Backend: Vulkan API
```
